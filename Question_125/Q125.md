# Question 125: Valid Palindrome

## Problem Description
Given a string `s`, determine if it is a palindrome, considering only alphanumeric characters and ignoring cases.

## Step-by-Step Explanation

1. **Initialize Pointers:**
   - Use two pointers, `left` and `right`, to traverse the string from both ends.

2. **Skip Non-Alphanumeric Characters:**
   - Move the `left` pointer forward and the `right` pointer backward until they point to alphanumeric characters.

3. **Compare Characters:**
   - Convert both characters to lowercase and compare them.
   - If the characters are not equal, return `false`.

4. **Continue Traversal:**
   - Increment the `left` pointer and decrement the `right` pointer.

5. **Return Result:**
   - If all characters match, return `true`.

## Code Implementation

```java
class Solution_Q125 {
    public boolean isPalindrome(String s) {
        int left = 0, right = s.length() - 1;
        while (left < right) {
            while (left < right && !isAlphaNum(s.charAt(left))) left++;
            while (left < right && !isAlphaNum(s.charAt(right))) right--;

            char a = toLower(s.charAt(left));
            char b = toLower(s.charAt(right));
            if (a != b) return false;

            left++;
            right--;
        }
        return true;
    }

    private boolean isAlphaNum(char c) {
        return (c >= 'a' && c <= 'z') ||
               (c >= 'A' && c <= 'Z') ||
               (c >= '0' && c <= '9');
    }

    private char toLower(char c) {
        return (c >= 'A' && c <= 'Z') ? (char)(c - 'A' + 'a') : c;
    }
}
```
