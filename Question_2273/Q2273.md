# Question 2273: Remove Anagrams

## Problem Description
You are given a list of words. Remove all consecutive strings that are anagrams of each other, leaving only the first occurrence of each group of anagrams.

## Step-by-Step Explanation

1. **Understand the Problem:**
   - Two words are anagrams if they contain the same characters in the same frequency but in any order.
   - Consecutive anagrams in the list should be removed, keeping only the first occurrence.

2. **Use a Signature for Anagrams:**
   - For each word, sort its characters to create a "signature" that uniquely identifies its anagram group.

3. **Iterate Through the Words:**
   - Maintain a variable `lastSig` to store the signature of the last added word.
   - For each word, compute its signature and compare it with `lastSig`.
   - If the signature is different, add the word to the result list and update `lastSig`.

4. **Return the Result:**
   - Return the list of words after removing consecutive anagrams.

## Code Implementation

```java
class SolutionQ2273 {
    public java.util.List<String> removeAnagrams(String[] words) {
        java.util.ArrayList<String> result = new java.util.ArrayList<>();
        String lastSig = null;

        for (String w : words) {
            String sig = sortString(w);
            if (lastSig == null || !sig.equals(lastSig)) {
                result.add(w);
                lastSig = sig;
            }
        }
        return result;
    }

    private String sortString(String s) {
        char[] arr = s.toCharArray();
        java.util.Arrays.sort(arr);
        return new String(arr);
    }
}
```
