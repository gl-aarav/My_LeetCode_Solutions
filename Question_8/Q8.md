# Question 8: String to Integer (atoi)

## Problem Description
Implement the `myAtoi` function, which converts a string to a 32-bit signed integer (similar to the C/C++ `atoi` function).

The function discards as many whitespace characters as necessary until the first non-whitespace character is found. Then, starting from this character, it takes an optional initial plus or minus sign followed by as many numerical digits as possible and interprets them as a numerical value.

The function does not handle invalid input gracefully and stops processing as soon as an invalid character is encountered.

## Approach
1. **Skip Leading Whitespaces**:
   - Iterate through the string to skip all leading spaces.

2. **Handle Sign**:
   - Check if the next character is a '+' or '-' to determine the sign of the number.

3. **Convert Characters to Digits**:
   - Iterate through the string, converting valid numerical characters to digits.
   - Stop processing when encountering a non-digit character.

4. **Handle Overflow**:
   - Check if the number exceeds the range of a 32-bit signed integer and return the appropriate boundary value.

## Code
```java
class Solution {
    public int myAtoi(String s) {
        int n = s.length();
        int i = 0;
        while (i < n && s.charAt(i) == ' ') i++;
        int sign = 1;
        if (i < n) {
            char c = s.charAt(i);
            if (c == '+') { sign = 1; i++; }
            else if (c == '-') { sign = -1; i++; }
        }
        long num = 0;
        int max = Integer.MAX_VALUE;
        int min = Integer.MIN_VALUE;
        while (i < n) {
            char c = s.charAt(i);
            if (c < '0' || c > '9') break;
            int d = c - '0';
            num = num * 10 + d;
            long val = sign == 1 ? num : -num;
            if (val > max) return max;
            if (val < min) return min;
            i++;
        }
        num = sign == 1 ? num : -num;
        if (num > max) return max;
        if (num < min) return min;
        return (int) num;
    }
}
```
