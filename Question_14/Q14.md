# Question 14: Longest Common Prefix

## Problem Description
Write a function to find the longest common prefix string amongst an array of strings.

If there is no common prefix, return an empty string `""`.

## Approach
1. **Start with the First String**:
   - Assume the first string is the prefix.

2. **Iterate Through Strings**:
   - Compare the prefix with each string in the array.
   - Reduce the prefix length until it matches the start of the current string.

3. **Return Result**:
   - Return the prefix after processing all strings.

## Code
```java
class Solution {
    public String longestCommonPrefix(String[] strs) {
        if (strs == null || strs.length == 0) return "";
        String prefix = strs[0];
        for (int i = 1; i < strs.length; i++) {
            while (!strs[i].startsWith(prefix)) {
                if (prefix.isEmpty()) return "";
                prefix = prefix.substring(0, prefix.length() - 1);
            }
        }
        return prefix;
    }
}
```
