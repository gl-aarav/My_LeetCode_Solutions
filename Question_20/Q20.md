# Question 20: Valid Parentheses

## Problem Description
Given a string `s` containing just the characters `(`, `)`, `{`, `}`, `[` and `]`, determine if the input string is valid.

An input string is valid if:
1. Open brackets must be closed by the same type of brackets.
2. Open brackets must be closed in the correct order.
3. Every close bracket has a corresponding open bracket of the same type.

## Approach
1. **Use a Stack**:
   - Push open brackets onto the stack.
   - For closing brackets, check if the top of the stack matches the corresponding open bracket.

2. **Check for Validity**:
   - If the stack is empty when a closing bracket is encountered, the string is invalid.
   - After processing all characters, the stack should be empty for the string to be valid.

3. **Edge Cases**:
   - Strings with odd lengths are automatically invalid.
   - Strings with unmatched brackets are invalid.

## Code
```java
public class Solution {
    public boolean isValid(String s) {
        if (s.length() % 2 != 0) return false;

        char[] stack = new char[s.length()];
        int top = -1;

        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);

            if (c == '(' || c == '{' || c == '[') {
                stack[++top] = c;
            } else {
                if (top < 0) return false;
                char t = stack[top--];
                if (c == ')' && t != '(') return false;
                if (c == '}' && t != '{') return false;
                if (c == ']' && t != '[') return false;
            }
        }

        return top == -1;
    }
}
```
