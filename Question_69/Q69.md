# Question 69: Sqrt(x)

## Problem Description
Given a non-negative integer `x`, compute and return the square root of `x`. Since the return type is an integer, the decimal digits are truncated, and only the integer part of the result is returned.

## Step-by-Step Explanation

1. **Understand the Problem:**
   - The task is to compute the integer square root of a non-negative integer `x`.
   - The result should be truncated to an integer.

2. **Newton's Approximation:**
   - Use an iterative method to approximate the square root.
   - Start with an initial guess `r = x`.
   - Update the guess using the formula:
     r = (r + x / r) / 2
   - Repeat until r * r <= x.

3. **Handle Small Values:**
   - If `x < 2`, return `x` directly as the square root.

4. **Return the Result:**
   - Return the integer value of `r` after the approximation.

## Code Implementation

```java
class Solution_Q69 {
    public int mySqrt(int x) {
        if (x < 2) 
            return x;

        long r = x; 
        while (r * r > x) {
            r = (r + x / r) / 2;
        }
        return (int) r;
    }
}
