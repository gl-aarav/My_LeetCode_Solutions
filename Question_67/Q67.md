# Question 67: Add Binary

## Problem Description
Given two binary strings `a` and `b`, return their sum as a binary string.

## Step-by-Step Explanation

1. **Initialize Pointers and Carry:**
   - Use two pointers `i` and `j` to traverse the binary strings `a` and `b` from the end.
   - Initialize a `carry` variable to handle sums greater than 1.

2. **Iterate Through Strings:**
   - While there are still digits left in `a` or `b`, or there is a carry:
     - Extract the current digits `da` and `db` from `a` and `b` (or 0 if out of bounds).
     - Compute the sum of `da`, `db`, and `carry`.
     - Store the least significant bit of the sum in the result array.
     - Update the carry to the most significant bit of the sum.

3. **Handle Leading Zeros:**
   - Remove any leading zeros from the result array.

4. **Return the Result:**
   - Convert the result array to a string and return it.

## Code Implementation

```java
class SolutionQ67 {
    public String addBinary(String a, String b) {
        int i = a.length() - 1;
        int j = b.length() - 1;
        int carry = 0;
        char[] res = new char[Math.max(a.length(), b.length()) + 1];
        int k = res.length - 1;

        while (i >= 0 || j >= 0 || carry != 0) {
            int da = (i >= 0) ? (a.charAt(i) - '0') : 0;
            int db = (j >= 0) ? (b.charAt(j) - '0') : 0;
            int sum = da + db + carry;
            res[k] = (char) ('0' + (sum & 1));
            carry = (sum >> 1);
            i--;
            j--;
            k--;
        }

        while (k >= 0) {
            res[k] = '0';
            k--;
        }

        int start = 0;
        while (start < res.length - 1 && res[start] == '0') start++;
        return new String(res, start, res.length - start);
    }
}
```
