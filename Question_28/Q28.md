# Question 28: Find the Index of the First Occurrence in a String

## Problem Description
Given two strings `haystack` and `needle`, return the index of the first occurrence of `needle` in `haystack`, or `-1` if `needle` is not part of `haystack`.

## Step-by-Step Explanation

1. **Understand the Problem:**
   - The task is to find the starting index of the first occurrence of the substring `needle` in the string `haystack`.
   - If `needle` is not found, return `-1`.

2. **Use Built-in Functionality:**
   - Java provides a built-in method `indexOf` in the `String` class that can be used to find the first occurrence of a substring.

3. **Return the Result:**
   - Directly return the result of `haystack.indexOf(needle)`.

## Code Implementation

```java
class Solution_Q28 {
    public int strStr(String haystack, String needle) {
        return haystack.indexOf(needle);
    }
}
```
